# telecom-analysis
final project
ConnectaTel – Análisis de Clientes
Objetivo del proyecto

Analizar los datos de clientes de ConnectaTel para identificar patrones de uso, detectar problemas en los datos y segmentar usuarios por comportamiento y edad. El objetivo es generar insights útiles para mejorar los planes del servicio.

Datasets utilizados
users: información de clientes (edad, ciudad, plan, fechas)
usage: registros de uso (mensajes, llamadas, minutos)
plans: información de los planes disponibles
